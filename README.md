# wallpaper-scraper
A small script to download wallpapers from threads on /wg/

## Requirements
**curl** and **jq**

```
Usage: paperscraper.sh
      -l (show list of threads), 
      -u (update catalog and threads), 
      -d <specify directory for files>, 
      -t <thread number to download>, 
      -h [help]
```
